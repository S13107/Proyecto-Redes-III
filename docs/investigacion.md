# Investigación: Seguridad en Redes y Firewalls

## 1. Tipos de Firewalls
Para este proyecto hemos analizado los siguientes tipos:
* **Firewall de Red (Stateful):** Filtra tráfico basado en el estado de las conexiones, IP y puertos.
* **Firewall de Aplicación (Proxy):** Realiza una inspección profunda en la capa 7, analizando el contenido específico del tráfico web.
* **Diferencia Clave:** El de red se centra en el transporte, mientras que el de aplicación entiende el protocolo y puede bloquear ataques específicos de software.

## 2. Principios CIA
Toda nuestra configuración se basa en el cumplimiento de estos tres pilares:
* **Confidencialidad:** Garantizar que los datos solo sean accesibles por personal autorizado mediante cifrado y VPN.
* **Integridad:** Asegurar que la información no sea alterada durante el tránsito.
* **Disponibilidad:** Mantener los servicios operativos mediante la flexibilidad y gestión centralizada de SD-WAN.
