# Investigación: Seguridad en Redes y Firewalls

## 1. Tipos de Firewalls
Para este proyecto hemos analizado los siguientes dispositivos:
**Firewall de Red:** Actúa en las capas 3 y 4 del modelo OSI, filtrando por IP y puerto[cite: 14].
**Firewall de Aplicación (Proxy):** Realiza una inspección profunda en la capa 7, analizando el contenido específico del tráfico web[cite: 14, 15].
**Diferencia Clave:** Mientras el de red se centra en el origen y destino, el de aplicación entiende el protocolo y puede bloquear ataques específicos de software[cite: 15].

## 2. Principios CIA
Toda nuestra configuración se basa en:
**Confidencialidad:** Garantizar que los datos solo sean accesibles por personal autorizado[cite: 6, 26].
**Integridad:** Asegurar que la información no sea alterada durante el tránsito[cite: 6, 26].
**Disponibilidad:** Mantener los servicios operativos mediante redundancia en SD-WAN[cite: 6, 26, 30].
